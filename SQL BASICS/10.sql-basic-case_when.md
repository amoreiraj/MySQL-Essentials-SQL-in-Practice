# Conditional logic with CASE WHEN in SQL

The `CASE WHEN` statement allows you to add **conditional logic** to SQL queries.  
It works like an `IF / ELSE` statement and is commonly used to **create categories**, **labels**, and **derived columns**.

All examples in this tutorial use the **hotel_reservations** table.

---

## What this tutorial covers

- What `CASE WHEN` is
- Basic `CASE WHEN` syntax
- Using `CASE WHEN` to create new columns
- Categorizing numeric values
- Using `CASE WHEN` with real booking data
- Common mistakes to avoid

---

## What is CASE WHEN?

`CASE WHEN` evaluates conditions **row by row** and returns a value when a condition is met.

Think of it as:

- **IF this condition is true → return this**
- **ELSE IF another condition is true → return that**
- **ELSE → return a default value**

---

## Basic CASE WHEN syntax

```sql
CASE
    WHEN condition THEN result
    ELSE default_result
END
```
